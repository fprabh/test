PLC FINAL PROJECT

BY: PRABH DHALIWAL

Introduction

An Elevator control was introduced as the final project for PLC course. The goal of the project was to master PLC's using state methods and have a clear understanding of working of PLC's scan time, asynchronous updating etc. Biggest challenge was to simulate the operation of a real elevator using the available lab equipment. Thankfully, I was able to get as close as possible by having some extra features and fault detections.

Final Approach

PLC Goes To Run Mode

When PLC goes to run mode during the first scan all states are set to 0, every floor call is cleared, emergency is cleared, Floor_Processing_Counter is set to 0 and initialization is performed. However if there were some faults from last time that were not cleared would remain be on until cleared.

Initialization

Initialization can be done either by switching PLC from program to run mode or by pressing the reset button if a major fault or an emergency stop was pressed. During initialization elevator goes to the first floor regardless of any floor calls that were made during that time. Door does not open when elevator reaches first floor unless Door Open pushbutton is pressed or an emergency has occurred. If there where any floor calls made during initialization, it saves the call and goes directly to the called floor once it reaches first floor.
